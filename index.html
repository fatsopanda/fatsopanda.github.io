Graphics programming
I started to get more into graphics programming during the spring 2016.
Firstly I began to experiment with shaders in Unity3D and soon learned that there's a whole another world within the possibilities you can create with shaders.

My thirst for more knowledged drove me to teach myself to code OpenGL and C++. In the learning process I started to enjoy coding more and more with C++ so I decided to start building my own rendering engine.

It's still currently work in progress, but I'm hoping to have it finished this fall. Currently it does deferred shading with shadow mapping and some post-processing (HDR, Bloom, SSAO). I'd like to add cascaded shadow maps into it and possibly collision detection so that I can create a small playable game with it.
